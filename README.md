# 🔮 Poções e Soluções

Projeto de uma loja temática de poções com interface web, contendo catálogo de produtos e painel administrativo para gerenciamento.

---

## 🚀 Tecnologias utilizadas

### Frontend
- React.js
- React Router DOM
- Axios
- CSS3 (estilização global e componentes)

### Backend
- Node.js
- Express.js
- CORS
- Banco de dados (conforme implementação: JSON / MongoDB / etc.)

---

## 📁 Estrutura do projeto
- Frontend;
- Backend.

---

## ⚙️ Como executar o projeto

# 🔧 Backend

### 1. Acesse a pasta do backend
```bash
cd backend
```

### 2. Instale as dependências
```bash
npm install
```

### 3. Inicie o servidor
```bash
npm start
```

# 💻 Frontend
### 1. Acesse a pasta do frontend
```bash
cd frontend
```

### 2. Instale as dependências
```bash
npm install
```

### 3. Inicie o projeto
```bash
npm run dev
```
